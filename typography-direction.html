Icon System Decision

Decision: NO icon library. Formality 7 + Richness 10 meets the "Formality 7+ AND Richness 8+" threshold for skipping icons entirely.

Icons cheapen luxury aesthetics. RH (Restoration Hardware), Poliform, and B&B Italia — all of which operate at this tier — use zero icon libraries. Visual communication is handled entirely by typography hierarchy and photography.

UI indicators (arrows, menu, close) are implemented as pure CSS or minimal inline SVG <path> tags copied from a single trusted source at build time — never a library load.
