## Analysis

### 1. What is the "local baseline"?

The local and regional competitors (Set Interieur, Laura Hindriks, Studio &Mar, Lux Interieurbouw, Su Casa) set a relatively accessible bar:

- **Visual quality is inconsistent.** Most use a single static hero photo with no video. Studio &Mar and Laura Hindriks are the strongest visually; Su Casa and Set Interieur are functional but not inspiring.
- **Social proof is thin or absent.** The best local example (Studio &Mar) has 14 named Google reviews prominently displayed. Most competitors have no reviews at all on their homepage. This is an easy gap to exploit.
- **International scope is non-existent locally.** None of the regional competitors show international projects. By Shakir's Dubai, Barcelona, Malaga, and Antwerp portfolio is already above the local market ceiling.
- **CTAs are passive.** "Bekijk onze projecten" and "Lees meer" dominate. Nobody in the region is using a strong consultation-qualification CTA or WhatsApp.
- **Typography is middling.** Cormorant + warm tones (Studio &Mar) is the strongest local execution but it's still boutique-scale. No one in the region is operating at the visual sophistication of the national leaders.

**Local baseline score: 5–6/10.** The regional market is not well-designed. A well-executed Astro build with professional photography and Dutch copy would immediately lead the local search results.

---

### 2. What do the national leaders do differently?

The top tier (Piet Boon, Eric Kuster, Kolenik, Lagrand) separates itself on five dimensions:

1. **Scale of social proof.** Press logos (Elle Decoration, HOOG.design, Stijlvol Wonen, Dezeen) replace or supplement Google reviews. Named brand-name clients (Four Seasons, Rosewood, Camp Nou) signal exclusivity. By Shakir has no press coverage yet — this is the single biggest gap relative to the national leaders.

2. **International project locations are explicitly named.** Not just "we work internationally" — specific cities and countries appear in project thumbnails and case study titles. Kolenik lists Grand Cayman, Ibiza; Piet Boon lists 46 countries. By Shakir has this material (Dubai, Barcelona, Malaga) but has not been displaying it prominently enough.

3. **Lifestyle brand extension.** Eric Kuster and Kolenik have moved beyond services into products (furniture, fragrances, cabinets). This is aspirational for By Shakir's collections offering — though collections were dropped for this build, the positioning logic is sound long-term.

4. **Restraint in CTAs.** No urgency, no "bel nu," no price anchoring. The CTA architecture at the national level is entirely soft — "Make an Appointment," "Book," or no explicit CTA at all on the homepage. The work and reputation do the asking.

5. **Whitespace as a design statement.** Piet Boon's cream palette with generous whitespace signals budget via breathing room. The national leaders understand that what you don't show is as important as what you do.

---

### 3. What mistakes do the worst competitors make?

Drawn from the weaker local examples and common failure patterns visible across the research:

1. **Copy-heavy, image-light homepages.** Su Casa leads with interrogative headlines and paragraphs of text. For an interior designer, the visual IS the pitch. Any homepage that needs to convince with words before images is positioning incorrectly.

2. **No differentiation between service levels or client types.** Most local competitors list "interieuradvies" and "interieurontwerp" without helping the visitor understand which is right for them — or whether their project qualifies. This causes the wrong leads to inquire and the right leads to leave.

3. **Missing WhatsApp.** Studio &Mar, Lagrand, Piet Boon, and most of the local competitors have no WhatsApp contact. For a Dutch audience in 2026, WhatsApp is the lowest-friction contact method. Its absence in the luxury segment is understandable (brand positioning), but for By Shakir's "warm, approachable" personality it is a missed opportunity.

4. **No process explanation.** The best local site (Studio &Mar) mentions "vrijblijvende offerte" but doesn't explain what happens after contact. High-ticket buyers (€50K+ interior projects) need to understand the steps before committing to inquiry. The 3D design process is By Shakir's strongest answer to this — it needs to be explained on the site, not assumed.

5. **Thin project pages.** Every local competitor shows 4–8 photos per project with minimal or no copy. The national leaders (Piet Boon, Lagrand) use full case studies with location, brief, scope, and client narrative. Depth is a trust signal — a 30-photo project page with a client story outperforms a 6-photo grid every time at the luxury price point.

### Studio Piet Boon (9/10) — pietboon.com

**Section order:**
1. Fixed sticky header (logo left, horizontal nav right, search icon)
2. Full-bleed hero slider — 780px desktop / 600px mobile
3. Constrained content area (max-width: 1150px)
4. Footer (full-width)
5. Off-canvas slide panel (fixed right, slides in from edge)

**Colors (CSS variables):**
- `--e-global-color-primary: #2D2D2D` — dark charcoal (text)
- `--e-global-color-text: #E9EAE3` — warm off-white (body background)
- `--e-global-color-accent: #E1E0DA` — pale taupe (hero section bg)
- `--e-global-color-secondary: #446679` — muted slate blue (buttons)
- Body background: `#E9EAE3` (warm, not cold white — this matters)
- Grayscale ramp: `#F5F5F4 → #D3D1C8 → #C2BEB3 → #B4B0A5 → #8F8B82 → #60584E → #4C4A44 → #343434`

**Typography:**
- Single font family: `"Avenir", Sans-serif` — used for everything
- Hero heading: 40px, font-weight: 400 (lighter weight = luxury signal)
- Body: 16px / 31px line-height (generous leading)
- h3: 21px / 29px
- Nav items: 14–16px, weight 400
- Letter-spacing: generous throughout

**Full-bleed sections:**
- Hero slider: edge-to-edge, padding `0 0 90px 0` only at bottom
- Header: full-width
- Off-canvas panel: `position: fixed; top: 30%; right: -250px` — slides in from right edge

**Asymmetric / overlapping elements:**
- Off-canvas panel: `right: -250px` default, slides to visible on trigger — a persistent editorial panel
- Decorative icon overlap: `margin: -60px 0 0 0` (negative top margin pulls element up over previous section)
- Element transform: `--e-transform-translateX: -82px; --e-transform-translateY: 24.5px` (slight offset positioning)
- Hero text: `position: absolute`, `padding: 70px` — text overlays full-bleed image
- Text shadow on all hero headings: `text-shadow: 9px 3px 8px rgb(0 0 0 / .33)`

**Animation:**
- Swiper slider: `transform: translate3d(0,0,0)` (GPU-accelerated)
- Off-canvas panel: CSS transition on `right` property
- Menu open: `--n-menu-open-animation-duration: 300ms`
- Scroll behavior: `scroll-behavior: smooth` with reduced-motion support

**Decorative elements:**
- No decorative SVG shapes or background patterns — restraint IS the design
- Section dividers: `border-bottom: 2px solid #fff0` (faint)
- Background images defined via CSS repeater: `background-size: cover; background-position: 50%`

**Section order:**
1. Sticky header (logo left, nav right, search + socials)
2. Full-bleed hero banner slider — `calc(100vh - 111px)`, max-height: 900px
3. Intro text section (constrained): "Waar consumenten, architecten én specialisten samenkomen"
4. "Gerealiseerd door specialisten" heading
5. Project grid carousel — `repeat(auto-fill, minmax(230px, 1fr))`, 4:5 aspect ratio cards
6. Partner logo slider (full-bleed carousel, `mix-blend-mode: multiply` on logos)
7. Newsletter section
8. Footer

**Colors:**
- Gold/primary: `#bf942a` (links, buttons, accents, nav dots, title punctuation)
- Dark: `#1a1a1a`
- Body text: `#666`
- Background: `#f8f8f8` / `#efe9e0` (warm beige)
- Border: `#eeeef0`
- Dark green: `#01311f` (styleRow backgrounds — a deeper accent)

**Typography:**
- Headings: `"Poppins", sans-serif`, weight 600
- Body: `"Inter", sans-serif`, weight 400
- `.bigTitle`: `clamp(1.75rem, 1.346rem + 3vw, 4rem)` — fluid, jumps from ~28px to 64px
- `.subTitle`: `clamp(1.25rem, 1.05rem + 1vw, 2.25rem)`
- **Distinctive detail:** All `.bigTitle` headings get `:after { content: "."; color: #bf942a }` — gold period at end of every major heading
- Body: 1rem / 16px base

**Full-bleed sections:**
- Hero banner: full-width, near-full-viewport height
- Partner logo strip: full-width carousel
- Newsletter section: full-width

**Constrained sections:**
- All content inside `.container` at 1500px / 1250px / 1100px (responsive)
- Project grid: inside container at max 1500px

**Asymmetric / overlapping elements:**
- Hero banner grid: `grid-template-columns: 2fr 1fr` — asymmetric split between image and content
- Banner layering: image (z-index: 1), overlay (z-index: 3), nav buttons (z-index: 2)
- `.styleRow__content:before` pseudo-element creates full-height background block (z-index: 1)
- Named grid areas: `grid-template-areas: "img1 txt" / "img1 img2"` (text+image interlocking layout)

**Animation:**
- SVG slogan in header: staggered fade-in across 5 groups (delays: 0.1s → 0.5s)
- `@keyframes fade-in-animation { 0%: opacity 0; 100%: opacity 1 }`
- Lazy load images: `.lazy` opacity 0 → 1 on `.loaded` (0.3s transition)
- Dropdown: `transform: translateY(-15px)` → `translateY(0)`, fade in, 0.3s ease
- Mobile nav: `transform: translateX(-100%)` → `translateX(0)` slide-in

**Decorative elements:**
- **Gold period after every heading** via `:after { content: "." }` — most ownable visual detail
- Animated SVG groups in header (brand slogan appears letter by letter)
- `mix-blend-mode: multiply` on partner/sponsor logos (blends onto backgrounds cleanly)
- Dot navigation for carousel: 15px circles, active = `#bf942a`
- Extensive use of `clamp()` for all spacing — nothing is hard-coded

- Logo: white on dark header

**Typography:**
- Not explicitly declared (Squarespace webfont loader) — likely a geometric sans-serif based on brand aesthetic
- Navigation: ALL CAPS, letter-spaced — a deliberate luxury signal
- Animation on font load: `@keyframes fonts-loading { 0%, 99% { color: transparent } }` — text invisible until webfonts loaded (prevents FOUT)

**Full-bleed sections:**
- Hero: full-bleed with `2vw` page padding
- Footer: full-width background

**Layout system:**
- 24-column grid (desktop, 768px+), 8-column (mobile) — high resolution grid
- `--sqs-site-gutter: 4vw` (desktop), `--sqs-mobile-site-gutter: 6vw` (mobile)
- Row gap: 11px, column gap: 11px — tight, controlled
- Max width: `var(--sqs-site-max-width, 1500px)`

**Asymmetric / overlapping elements:**
- Hero: image `z-index: 2`, text block overlaps image partially — desktop text shifts to `grid-area: 1/9/3/19`
- Portfolio gallery: `margin-right: -20px` (negative gutter collapse for flush edge-to-edge row)
- Gallery scaling: `--row-height-scaling-factor: 0.0215` (proportional heights)

**Animation:**
- Global: fade, 1.5s duration, 0.6s delay, ease, "detailed" complexity — slow, deliberate, luxury-paced
- Header: `backdrop-filter: blur(12px)` — glassmorphism effect on scroll
- Font loading: flash prevention via transparent animation

**Decorative elements:**
- Hamburger: pure CSS (`.top-bun`, `.patty`, `.bottom-bun` — 1px height lines), minimal
- Backdrop blur on header background — modern premium treatment
- No decorative shapes or patterns — portfolio images are all the decoration

## What the Lowest-Rated Competitors Do Wrong

**Set Interieur (4/10) and Lux Interieurbouw (4/10):**

| Problem | Set Interieur | Lux Interieurbouw |
|---|---|---|
| **Hero** | Single static photo, doesn't use full viewport height — "cropped to the middle, feels weird on PC" | 3-slide commercial carousel with copper gradient buttons — feels like a tradesman's site |
| **Color** | Monochromatic, safe, zero personality | Copper gradient CTAs signal cheap/commercial, not luxury |
| **Typography** | No hierarchy differentiation, body-weight headings | Stats-forward (234 projects / 7682 hours) feels corporate — wrong emotional register |
| **Animation** | None | "Old animations" — CSS transitions that feel dated (bounce, wipe effects) |
| **Density** | Restrained but not intentional — just sparse | Over-dense: 50-year history + metrics + carousel + social links all compete |
| **Mobile** | Not designed for the full desktop viewport | Social links to Instagram/Facebook/TikTok signal B2C trades, not luxury |
| **Portfolio** | Not portfolio-heavy enough on the homepage | Kitchen/wardrobe focus feels functional, not aspirational |

**Root cause:** Both treat the homepage as an information delivery system rather than an emotional experience. They answer "what do we do?" before the visitor has been made to feel anything.

## Summary: Top 5 Design Decisions to Incorporate

| # | Decision | Source | How to apply for By Shakir |
|---|---|---|---|
| 1 | **Fluid typography with `clamp()`** | Best Interiors | All type scales fluid — no fixed px breakpoint jumps. Hero heading from ~32px mobile to ~64px desktop. |
| 2 | **Warm off-white body background** | Piet Boon (`#E9EAE3`) | Never use pure #FFFFFF. Body background should be a warm cream/beige — signals luxury, reduces eye fatigue, matches interior photography. |
| 3 | **4:5 aspect ratio for project cards** | Best Interiors | Portrait-format cards for the project grid — more cinematic, fills screen height, shows interiors better than 16:9. |
| 4 | **ALL CAPS, letter-spaced navigation** | Kolenik | Nav items in uppercase with generous letter-spacing — one of the clearest luxury signals available for near-zero effort. |
| 5 | **Backdrop blur (`blur(12px)`) on scrolled header** | Kolenik | Header transitions from transparent to blurred background on scroll — glass effect, modern premium feel, eliminates the harsh header line. |

## CSS Layout Patterns That Make the Best Ones Look Professional

```css
/* 1. Fluid type — never hard-code heading sizes */
.heading-xl { font-size: clamp(2rem, 1.5rem + 3vw, 4.5rem); }
.heading-lg { font-size: clamp(1.5rem, 1.2rem + 2vw, 3rem); }

/* 2. Warm body background — not pure white */
body { background-color: #F2EDE6; } /* warm cream, similar to Piet Boon's #E9EAE3 */

/* 3. Full-bleed hero with constrained content inside */
.hero { width: 100%; height: calc(100vh - var(--header-height)); }
.content-container { max-width: 1400px; margin: 0 auto; padding: 0 clamp(1rem, 4vw, 5rem); }

/* 4. Portrait project cards — 4:5 ratio */
.project-card { aspect-ratio: 4 / 5; overflow: hidden; }
.project-card img { width: 100%; height: 100%; object-fit: cover; }

/* 5. Glassmorphism header */
.header.scrolled {
  backdrop-filter: blur(12px);
  -webkit-backdrop-filter: blur(12px);
  background: rgba(255, 255, 255, 0.7);
}

/* 6. Negative margin overlap for section transitions */
.overlap-up { margin-top: -4rem; position: relative; z-index: 2; }

/* 7. Asymmetric hero grid */
.hero-grid { display: grid; grid-template-columns: 2fr 1fr; }

/* 8. Text shadow for hero legibility (Piet Boon) */
.hero-heading { text-shadow: 9px 3px 8px rgb(0 0 0 / 0.33); }

/* 9. mix-blend-mode for logo/badge integration */
.partner-logo { mix-blend-mode: multiply; }

/* 10. Hero heading at lighter weight — looks more expensive */
.hero-heading { font-weight: 300; letter-spacing: 0.02em; }
```
