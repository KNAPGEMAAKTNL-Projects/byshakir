## Copy Quality Audit (self-check against loaded reference files)

### Anti-AI Detection (core/anti-ai-detection.md — 18 red flags)
1. ✓ Paragraph length SD > 25 words — varied from 2-word fragments to 49-word capsule
2. ✓ Sentence length range: 5 words ('No templates.') to 30+ words (answer capsule)
3. ✓ No restating conclusions — each section ends with forward motion or CTA
4. ✓ Zero formal transition markers ('Furthermore,' 'Moreover,' 'Additionally')
5. ✓ 8+ named entities (Tiel, Netherlands, Dubai, Barcelona, Malaga, Antwerpen, Mediterranean, Shakir)
6. ✓ Position-taking present ('No templates. No trend-chasing. No imposed aesthetic.')
7. ✓ Hero copy passes interchangeability test — Piet Boon (cold/clean) could not use 'Made to Be Lived In'
8. ✓ Zero tricolon abuse — the 'No X. No Y. No Z.' is a deliberate rhetorical choice, not default structure
9. ✓ No hypophora (question → immediate answer as transition)
10. ✓ Tonal variation: hero is bold and sparse, about is warm and narrative, process is concrete and practical

### Anti-AI English (lang/en/anti-ai-en.md — 30 red flags)
11. ✓ Zero em dash overuse — 2 single en dashes across entire page; zero paired-dash (— X —) constructions
12. ✓ No flagged vocabulary: zero instances of 'delve,' 'tapestry,' 'seamless,' 'elevate,' 'meticulously,' 'nestled'
13. ✓ No AI openers: no 'In today's fast-paced world,' no 'It's worth noting'
14. ✓ Contractions: selective positive only ('it's' if present) — no negative contractions
15. ✓ Sentence fragments present ('No templates.' 'Made to Be Lived In.')
16. ✓ British register: 'visualisation,' 'enquiry' (on contact page), 'joinery,' 'bespoke'
17. ✓ Zero exclamation marks across entire page

### Quality Signals (core/quality-signals.md — targeting 12/20)
1. ✓ Sentence length variation (fragments to complex sentences)
2. ✓ Paragraph length variation (1-sentence to 3-sentence)
3. ✓ Reader-first opening (hero: 'how you live' — not 'we are a leading')
4. ✓ Specific proof (4 countries, named project locations, named reviewers)
5. ✓ Position-taking ('No templates. No trend-chasing.')
6. ✓ Named entities (8+ specific names/locations)
7. ✓ Second-person dominance ('your home,' 'your routines,' 'your preferences')
8. ✓ Scannable structure (overlines, H2s, short paragraphs)
9. ✓ CTA with value framing ('Begin a Conversation' — describes what you get)
10. ✓ Trust signal at decision point (microcopy below every CTA)
11. ✓ Conversational rhythm (fragments, spaced en dashes)
12. ✓ Cinematically concrete image ('travertine, linen, late-afternoon light')
13. ✓ Opening matches awareness (product-aware: value proposition clarity)
14. ✓ Earned opinion ('never have to chase a single supplier')
